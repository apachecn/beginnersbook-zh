## 1.算术运算符

*   +加法运算符
*   - 减法运算符
*   *乘法运算符
*   /部门经营者
*   %模数运算符

### 算术运算符的例子

```
fun main(args: Array<String>) {

   val num1 = 101.99
   val num2 = 100.50
   var op: Double

   op = num1 + num2
   println("Addition: $op")

   op = num1 - num2
   println("Subtraction: $op")

   op = num1 * num2
   println("Multiplication: $op")

   op = num1 / num2
   println("Division: $op")

   op = num1 % num2
   println("Modulus: $op")
}
```

输出：

```
Addition: 202.49
Subtraction: 1.4899999999999949
Multiplication: 10249.994999999999
Division: 1.014825870646766
Modulus: 1.4899999999999949
```
